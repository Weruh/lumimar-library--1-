# Lumimar Library

Frontend-only project.

- Checkout runs fully in the browser.
- Orders are saved in `localStorage`.
- Download links are generated locally in the frontend-only flow.

## Project structure

- `frontend/`: Vite + React app

## Run locally

**Prerequisites:** Node.js 22+

1. Install dependencies:
   `npm install`
2. Start frontend only:
   `npm run dev`
